Experience / FRC Robotics

Sep 2022 - Jun 2024 / Oakville, ON

FRC Robotics Team Lead Programmer and Vision Specialist

Vision-system work for the 2024 CRESCENDO season, where the robot needed to detect and target orange Note game pieces.

Vision System

I led the initiative to bring machine vision back into the team workflow. The system used an Orange Pi as the processing unit and PhotonVision for game-piece detection.

  • Installed and configured a PhotonVision fork with Rockchip/RKNN support for Orange Pi hardware.
  • Used a pretrained YOLO model inside PhotonVision to detect CRESCENDO Notes in real time.
  • Output target information from the vision stack to the Java robot code.
  • Provided the vision module used by the drive-code lead for driver-assisted pickup integration.

Problem Solving

The main challenge was software compatibility. I spent time navigating PhotonVision versions, Orange Pi support, Chief Delphi resources, and GitHub forks until I found a setup that worked with the hardware available to the team.

Mentorship

Before graduating, I mentored five Grade 10 students on Linux basics, Orange Pi setup, PhotonVision installation, and connecting vision output to Java robot code. The next year, some of those students were able to build more advanced camera-based systems, including odometry work.

VacuumClone

A small native macOS utility that gives users a simple free button for quitting open apps.

Why I Built It

VacuumClone started as a quick response to a paid macOS utility that charged for a very small convenience: quitting open apps. I rebuilt the core idea as a free native app, partly as a joke and partly as a practical exercise in making a focused macOS tool.

What It Does

  • Provides a simple “quit all apps” style workflow for clearing a busy macOS session.
  • Uses Swift so the app fits naturally into macOS instead of feeling like a web wrapper.
  • Keeps the scope intentionally small: one convenience feature, no subscription, no extra platform around it.

Takeaway

The project is small, but it was useful practice for shipping a native desktop utility quickly: deciding what the app should not do, keeping the UI direct, and making the implementation match a very narrow user task.

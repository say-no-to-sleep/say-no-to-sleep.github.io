FRC Robotics Team Lead Programmer & Vision Specialist - School FRC Club

Sep 2022 - June 2024 (Oakville, ON)

As the Lead Programmer for our school's FRC Robotics Team, I spearheaded the initiative to integrate advanced machine vision capabilities for the 2024 CRESCENDO competition, which involved detecting and manipulating "Note" game pieces. This was a significant step for the team, as vision systems had not been utilized for several years.

Developing the Vision System:

My role encompassed the end-to-end design and implementation of a vision system aimed at enabling the robot to automatically locate and target game pieces. The technical solution involved:

  • Hardware & Software: Utilizing an Orange Pi single-board computer as the core processing unit. I successfully installed and configured a specialized fork of PhotonVision (by laviRZ) that included Rockchip (RKNN) support, which was crucial for leveraging the Orange Pi's processing power for machine learning tasks.
  • Object Detection: Implementing a pre-trained YOLO (You Only Look Once) model within the PhotonVision environment to detect the orange "Note" game pieces in real-time from the robot's camera feed.
  • Integration: The vision system was designed to output the location of detected Notes. This data was then provided to the main robot control system (programmed in Java), with the drive-to-target integration code being handled by our team captain. The intent was to create a driver-assisted targeting system, where a button press would trigger the robot to fine-tune its alignment with a detected Note for intake.
  • Problem-Solving: A key challenge was navigating the software landscape for the Orange Pi. I actively researched and utilized resources like the Chief Delphi forums and specific GitHub repositories to find compatible PhotonVision versions and guides, particularly the rknn-readme documentation for the laviRZ fork, which was instrumental.

During development and testing, the vision system demonstrated its capability to successfully detect and provide targeting data for the "Notes."

Mentorship and Knowledge Transfer:

Recognizing the need for continuity, I took the initiative to mentor a group of five Grade 10 students. I provided them with foundational knowledge on Linux basics, Orange Pi setup and usage, PhotonVision installation, and basic integration with the Java robot code. I created introductory materials to facilitate their learning. This early guidance and shared knowledge helped these students, with at least one, Ian, actively building upon this foundation. I was proud to learn that in the subsequent year, this group of students went on to independently implement even more complex systems, including camera-based odometry, showcasing the long-term value of this knowledge transfer.
